# 💸 BaseTip — On-Chain Tipping Protocol

> Send ETH tips to anyone, anywhere — with a message. 
> Fully on-chain on Base mainnet.

![Base](https://img.shields.io/badge/Built%20on-Base-0052FF?style=for-the-badge)
![Solidity](https://img.shields.io/badge/Solidity-0.8.20-363636?style=for-the-badge)

---

## 🎯 Problem It Solves

Creators, developers, and builders have no simple way 
to receive on-chain support from their community.

BaseTip fixes this — send ETH to any wallet with a 
message, instantly, with zero platform fees and full 
onchain transparency.

---

## ✨ Features

- 💸 **Instant Tips** — Send ETH to any wallet address
- 💬 **Message + Emoji** — Attach a note with every tip
- 📊 **Public Feed** — See all tips in real-time
- 👤 **Profile Stats** — Total received, sent, tip count
- 🏆 **Leaderboard** — Top supporters onchain
- 📡 **Onchain Events** — Every tip emits blockchain event
- 🔒 **No Platform Cut** — 100% goes to receiver
